# Design for CMS

A modern and responsive **CMS UI design** built using **HTML** and **Tailwind CSS**.  
This project focuses on clean layout, usability, and scalable design for content management systems.

## Features

- Responsive layout for all screen sizes
- Clean and modern UI design
- Reusable Tailwind CSS utility classes
- Optimized for CMS dashboards and content pages
- Fast loading and lightweight (no JS framework)

## Technologies Used

- **HTML5** – Semantic and accessible markup  
- **Tailwind CSS** – Utility-first CSS framework for rapid UI development
